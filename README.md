# Natural Language Understanding Assignment (NLU_A2)
This repository contains two Jupyter notebooks implementing key Natural Language Processing (NLP) tasks:
Q1: Learning Word Embeddings from IIT JODHPUR data.
Q2: Character-Level Name Generation using RNN Variants

### Requirements:
Make sure you have the following installed:
* Python (>= 3.8 recommended)
* Jupyter Notebook / JupyterLab

Required Python libraries:
- numpy
- pandas
- matplotlib
- seaborn
- scikit-learn
- torch 
- gensim (for Word2Vec comparison)

You can install dependencies using:
 ```pip install numpy pandas matplotlib seaborn scikit-learn torch gensim```

## How to Run
Step 1: Clone the Repository
```git clone [https://github.com/harishubha/NLU_A2]```

Step 2: Launch Jupyter Notebook
```jupyter notebook```

Step 3: Open and Run Files

Q1: Word Embeddings

Open:

*NLU_A2_Q1.ipynb*

This notebook includes:
Dataset preparation
Training word embeddings
Comparison with Word2Vec
Semantic analysis
Visualization of embeddings

Run all cells sequentially:
Kernel → Restart & Run All

Q2: Character-Level Name Generator

Open:

*NLU_A2_Q2.ipynb*

This notebook includes:
Dataset loading
RNN model implementation (variants)
Training and evaluation
Qualitative text generation

Run all cells sequentially:
Kernel → Restart & Run All

> If jupyter notebook method does not work, the *.ipynb* files can be downloaded and opened as Google Colab Notebooks. Then, run all cells sequentially by using Run All option.

Dataset Notes :
Ensure datasets (if external) are placed in the correct directory as referenced in the notebooks.
Update file paths inside the notebooks if necessary.
